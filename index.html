Virtual Chart Navigation Trainer

Educational Use Only

Welcome to the Virtual Chart Navigation Trainer — an interactive tool designed to help you practise fundamental coastal navigation techniques including dead reckoning, cross bearings, course planning, and more.

Please read before continuing:

  • This tool is intended for educational and training purposes only. It must not be used for real-world navigation, voyage planning, or safety-critical decisions.
  • Charts, landmarks, depths, and all geographic features are procedurally generated and do not represent any real-world location.
  • Icon placement, soundings, bearings, and labelling may contain inaccuracies or simplifications for the sake of the learning exercise.
  • Terminology and conventions used are approximate and may differ from official hydrographic publications (e.g. UKHO, NOAA).
  • Always refer to official, up-to-date charts and qualified instruction for any real navigation activity.

Feel free to contribute a fix or improvement on GitHub.
